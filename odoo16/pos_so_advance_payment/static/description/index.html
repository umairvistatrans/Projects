Create Sale Order, Delivery Address, Advance Payment from POS

This module allows the cashier to create Quotation/Sales Order directly from POS, collect Delivery Address of the customer and Register Advance Payment from POS.

Explore this module

Overview Key Features Setup & Configuration Screenshots

Overview

The System will Generate a Sales Order in the backend based on the Point of Sale (POS) to maintain Order Tracking.

Features

Activate the 'Enable Creating SO from POS' boolean setting in the POS Configuration.

Under Configuration, you can select the state which you want the SO to create and also enable the advance payment collection if needed.

In the Main POS screen, user will see a new button "Create Quotation". Cashier can click on it if incase he wants to register a Quotation/SO in backend for the current order from the POS.

Adding Customer and Product on POS screen is mandatory to create a Quotation.

Setup & Configuration

Configuration

Under Main menu Point of Sale > Dashboard > Particular Card -> Select 3 dots -> Edit, you need to enable 'Enable Creating SO from POS' boolean from POS Config.

Under Configuration, you can select the state which you want the SO to create and also enable the advance payment collection if needed.

Screenshots

Under Main menu Point of Sale > Dashboard > Particular Card -> Select 3 dots -> Edit, you need to enable 'Enable Creating SO from POS' boolean from POS Config.

Under Sale Order Configuration Section, you can select the state which you want the SO to create and also enable the advance payment collection if needed.

In the Main POS screen, user will see a new button "Create Quotation". Cashier can click on it if incase he wants to register a Quotation/SO in backend for the current order from the POS.

Selecting a Customer on the POS screen is mandatory to create a Quotation.

Upon clicking the Create Quotation Button on the Main POS Screen, a POP up will appear for Selecting or Creating a New Delivery Address.

The Existing Delivery Address will display the Parent Partner's Existing Delivery Address

The Create Sale Order POP up contains option for Creating a New address.

Cashier can directly collect advance payment from the customer from the POS screen.

User can also select the appropriate Journal from the available list.

Cashier can specify notes for the current Quotation/SO.

A sale order will be created in the backend with the delivery address specified during the order.

Invoices will be auto created against the SO and Respective Advance payment will be registered against the Invoice.

Another invoice with the remaining amount will be auto-created in open state.

A Popup displaying the Sales Order Reference will appear.
