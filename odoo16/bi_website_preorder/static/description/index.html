Pre-Order Out of Stock Products On Web Shop Odoo Apps

Pre-Order Website Product Odoo App enables businesses to offer pre-order functionality for their products through their website. With this Odoo app, businesses can accept pre-orders for products that are unavailable or out of stock. Users can set specific start and end dates, and amount types, specify pre-order min-max quantities, send mail options, and custom messages. Users can send email notifications to customers who have placed pre-orders automatically or manually, keeping them informed about the status of their orders and any updates regarding product availability or delivery timelines.

Features

Pre Order for Out of Stock Product

User can create pre orders for products that are currently out of stock.

Pre Order Product Label & Policy

User can see the "Available for Pre Order" label and policy on the website product page.

Pre Order Payment Type & Expiry Date

User can set payment type as a percentage of amount, and fixed amount, and also can set expiry for pre order.

Set Min-Max Quantity and Custom Messages

User can set minimum or maximum quantity and also can set custom messages for pre order.

Pre Order Confirmation Mail

The customer will get an email notification when the product is back in stock.

Override Default Pre-Order Configuration

User can set different conditions for each product with an override default pre order configuration option.
Demo Request Video Tutorial

Pre Order Menu

Under website > Configuration user can see the "Pre Order" menu.

Pre Order Configuration

On pre order configuration user need to enable the "Active" option to apply this configuration on website, user can select "Payment Type" as percentage of price, or fixed amount, also can set and show the expiry date for pre order product, show min and max quantity for Pre-order, select "Send Email option as Auto or Manual", if selected "Auto" then an email will be sent to customer once stock is available, if selected manual, user can send re order conformation email manually, User can also configure custom messages.

Pre Order Product

User need to enable the "Available for Pre-order" option under product, and only products are available for pre order, Under the "Pre-Order" tab user can see all configuration and can apply custom configuration for specific products by enabling "Override Default Pre-order Configuration" option.

Available for Pre Order

User can see "Available for Pre Order" label on product.

Pre Order

Customers can pre order products by clicking the "Pre Order" button on website, User also can see the "Pre Order Policy" on the product page.

User cans see added product into cart with 50 percentage of price as per configuration.

Send Pre Order Confirmation Email

Once pre order product is in stock user can send pre order confirmation email if user select manual option, Otherwise, email sent automatically.

Pre Order Confirmation Email

User can see pre order confirmation email, user can click the "View Order" button to see order.

User can pay the remaining amount of pre order by clicking the "Complete your payment" button.

User can see order amount is updated now user can confirm order and deliver order.
